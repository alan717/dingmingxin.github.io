github 地址: fzf-github 这是一个go语言写的工具，用于命令行的fuzzy finder 他的输入是一个数组，输出是用户的选择项。

我写的fzf工具

fzf-emacsbuffers

这个工具用于命令行，查看当前emacs所打开的buffer，选择后可以进入emacs并打开指定的buffer

注意：这个工具仅限于命令行下emacs 的daemon 模式

Example:

2016-05-05 2017-03-11 Fzf dingmingxin
